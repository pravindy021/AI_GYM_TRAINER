# AI Gym Trainer

The AI Gym Trainer is your smart, personalized fitness companion designed to guide, motivate, and transform your workouts. Powered by advanced algorithms, it analyzes your body type, fitness goals, and daily progress to create tailored exercise routines.

Personalized Workouts: Custom training plans adapt to your strength, stamina, and lifestyle.

Real-time Feedback: Tracks posture, reps, and form using sensors or camera input, ensuring safe and effective training.

Nutrition Guidance: Suggests balanced meal plans aligned with your fitness journey.

Progress Tracking: Monitors calories burned, muscle growth, and endurance improvements with detailed analytics.

Motivation Boost: Delivers reminders, challenges, and encouragement to keep you consistent.

Think of it as having a 24/7 digital coach who never gets tired, always adapts, and keeps you on track toward your fitness goals.

## Run in CLI mode

```bash
python app.py
```

## Run the GUI

```bash
python app.py --gui
```

## Run tests

```bash
python -m unittest discover -v
```
